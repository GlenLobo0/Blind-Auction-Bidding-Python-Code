# ☕ Coffee Machine Simulator

A Python-based console application that simulates the functionality of a coffee vending machine. The program allows users to order different coffee drinks, processes payments, manages ingredients, and generates machine reports.

## Features

- Order different coffee drinks:
  - Espresso
  - Latte
  - Cappuccino

- Resource Management
  - Tracks available water, milk, and coffee.
  - Prevents orders when resources are insufficient.

- Coin Processing System
  - Accepts user payment.
  - Calculates total inserted money.
  - Returns change when necessary.

- Transaction Validation
  - Verifies whether the payment is sufficient.
  - Cancels transactions if funds are insufficient.

- Machine Reports
  - Displays current ingredient levels.
  - Shows total profit earned.

- Admin Menu
  - Access machine statistics and reports.
  - Monitor machine resources.

## Technologies Used

- Dictionaries
- Loops
- Conditional Statements
- User Input Handling
